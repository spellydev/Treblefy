# Treblefy
Treblefy is a Magisk Module which aims to fix lag and other problems on most GSIs.

How to install:

Go on Magisk

Click on "Modules"

Choose "Treblefy.zip"

On the confirmation dialog, click "Yes"

Let it do it's thing

Reboot
